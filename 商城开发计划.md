# 🛒 电商商城开发计划

> 前后端分离架构 · Vue 3 + Node.js + MySQL
> 版本：v1.0 | 日期：2026-07-08

---

## 一、项目概述

### 1.1 项目定位
开发一套完整的企业级 B2C 电商平台，类似于淘宝/京东的简化版，覆盖用户购物全流程与商家运营管理。

### 1.2 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **前端（商城前台）** | Vue 3 + Vite + Pinia + Vue Router | 用户端体验优先 |
| **前端（后台管理）** | Vue 3 + Element Plus | 运营效率优先 |
| **后端** | Node.js + Express / Koa2 | 轻量灵活 |
| **数据库** | MySQL 8.0 | 关系型数据存储 |
| **缓存层** | Redis | Session、热点数据缓存 |
| **图片存储** | 阿里云 OSS / 七牛云 | 商品图片托管 |
| **构建部署** | Docker + Nginx | 生产环境部署 |

## 三、数据库设计

### 3.1 ER 关系概览

```
用户 (users) ──┬── 收货地址 (addresses)
               ├── 收藏商品 (favorites)
               └── 会员积分 (points)

商品 (products) ──┬── SKU规格 (skus) ──── 库存 (inventory)
                  ├── 商品图片 (images)
                  ├── 商品评价 (reviews)
                  └── 分类 (categories)

订单 (orders) ──┬── 订单明细 (order_items)
                ├── 支付记录 (payments)
                ├── 物流信息 (shipments)
                └── 售后/退款 (refunds)

营销 (promotions) ──┬── 优惠券 (coupons)
                    ├── 秒杀活动 (seckills)
                    └── 满减规则 (discount_rules)
```

### 3.2 核心数据表设计

#### 用户表 `users`
```sql
CREATE TABLE users (
  id           BIGINT PRIMARY KEY AUTO_INCREMENT,
  username     VARCHAR(50) UNIQUE NOT NULL,
  phone        VARCHAR(20) UNIQUE NOT NULL,
  email        VARCHAR(100) UNIQUE,
  password     VARCHAR(255) NOT NULL,        -- bcrypt 加密
  nickname     VARCHAR(50),
  avatar       VARCHAR(500),
  gender       TINYINT DEFAULT 0,            -- 0未知 1男 2女
  birthday     DATE,
  status       TINYINT DEFAULT 1,            -- 1正常 0禁用
  created_at   DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at   DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 收货地址表 `addresses`
```sql
CREATE TABLE addresses (
  id          BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id     BIGINT NOT NULL,
  receiver    VARCHAR(50) NOT NULL,
  phone       VARCHAR(20) NOT NULL,
  province    VARCHAR(50) NOT NULL,
  city        VARCHAR(50) NOT NULL,
  district    VARCHAR(50) NOT NULL,
  detail      VARCHAR(255) NOT NULL,
  is_default  TINYINT DEFAULT 0,
  created_at  DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 商品分类表 `categories`
```sql
CREATE TABLE categories (
  id          BIGINT PRIMARY KEY AUTO_INCREMENT,
  parent_id   BIGINT DEFAULT 0,               -- 0为顶级
  name        VARCHAR(100) NOT NULL,
  icon        VARCHAR(255),
  sort_order  INT DEFAULT 0,
  is_show     TINYINT DEFAULT 1,
  created_at  DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 商品表 `products`
```sql
CREATE TABLE products (
  id            BIGINT PRIMARY KEY AUTO_INCREMENT,
  category_id   BIGINT NOT NULL,
  brand_id      BIGINT,
  name          VARCHAR(200) NOT NULL,
  sub_title      VARCHAR(500),
  price         DECIMAL(10,2) NOT NULL,      -- 起售价
  original_price DECIMAL(10,2),               -- 原价
  cost          DECIMAL(10,2),                -- 成本价
  stock         INT DEFAULT 0,                -- 总库存
  sold_count    INT DEFAULT 0,                -- 销量
  view_count    INT DEFAULT 0,                -- 浏览量
  freight       DECIMAL(8,2) DEFAULT 0,       -- 运费
  is_on_sale    TINYINT DEFAULT 1,
  is_recommend  TINYINT DEFAULT 0,            -- 为您推荐
  is_new        TINYINT DEFAULT 0,            -- 新品
  description   TEXT,                         -- 详情（富文本）
  params        JSON,                         -- 参数规格JSON
  created_at    DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at    DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 商品SKU表 `skus`
```sql
CREATE TABLE skus (
  id           BIGINT PRIMARY KEY AUTO_INCREMENT,
  product_id   BIGINT NOT NULL,
  specs        JSON NOT NULL,                -- {"颜色":"红色","内存":"256G"}
  price        DECIMAL(10,2) NOT NULL,
  stock        INT DEFAULT 0,
  sku_code     VARCHAR(64) UNIQUE,
  image        VARCHAR(500),
  created_at   DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 商品图片表 `product_images`
```sql
CREATE TABLE product_images (
  id          BIGINT PRIMARY KEY AUTO_INCREMENT,
  product_id  BIGINT NOT NULL,
  url         VARCHAR(500) NOT NULL,
  is_main     TINYINT DEFAULT 0,             -- 主图
  sort_order  INT DEFAULT 0,
  created_at  DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 购物车表 `cart_items`
```sql
CREATE TABLE cart_items (
  id           BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id      BIGINT NOT NULL,
  product_id   BIGINT NOT NULL,
  sku_id       BIGINT,
  quantity     INT DEFAULT 1,
  created_at   DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at   DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 订单表 `orders`
```sql
CREATE TABLE orders (
  id              BIGINT PRIMARY KEY AUTO_INCREMENT,
  order_no        VARCHAR(32) UNIQUE NOT NULL,  -- 订单号
  user_id         BIGINT NOT NULL,
  address_id      BIGINT NOT NULL,
  total_amount    DECIMAL(12,2) NOT NULL,        -- 订单总额
  freight_amount  DECIMAL(8,2) DEFAULT 0,        -- 运费
  discount_amount DECIMAL(10,2) DEFAULT 0,      -- 优惠金额
  pay_amount      DECIMAL(12,2) NOT NULL,       -- 实付金额
  pay_method      VARCHAR(20),                   -- wechat/alipay/balance
  pay_time        DATETIME,
  status          TINYINT DEFAULT 10,           -- 10待支付 20已支付 30已发货 40已完成 50已取消
  remark          VARCHAR(500),
  created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at      DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 订单明细表 `order_items`
```sql
CREATE TABLE order_items (
  id            BIGINT PRIMARY KEY AUTO_INCREMENT,
  order_id      BIGINT NOT NULL,
  product_id    BIGINT NOT NULL,
  sku_id        BIGINT,
  product_name  VARCHAR(200) NOT NULL,
  sku_specs     JSON,                          -- 快照规格
  image         VARCHAR(500),
  price         DECIMAL(10,2) NOT NULL,
  quantity      INT NOT NULL,
  subtotal      DECIMAL(12,2) NOT NULL
);
```

#### 支付记录表 `payments`
```sql
CREATE TABLE payments (
  id              BIGINT PRIMARY KEY AUTO_INCREMENT,
  payment_no      VARCHAR(64) UNIQUE NOT NULL,
  order_id        BIGINT NOT NULL,
  user_id         BIGINT NOT NULL,
  amount          DECIMAL(12,2) NOT NULL,
  method          VARCHAR(20) NOT NULL,
  status          TINYINT DEFAULT 0,           -- 0待支付 1已支付 2已退款
  transaction_id  VARCHAR(128),                -- 第三方流水号
  paid_at         DATETIME,
  created_at      DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 优惠券表 `coupons`
```sql
CREATE TABLE coupons (
  id            BIGINT PRIMARY KEY AUTO_INCREMENT,
  name          VARCHAR(100) NOT NULL,
  type          TINYINT NOT NULL,             -- 1满减券 2折扣券 3无门槛券
  value         DECIMAL(10,2) NOT NULL,       -- 优惠金额或折扣率
  min_amount    DECIMAL(10,2) DEFAULT 0,       -- 满X元可用
  total_count   INT NOT NULL,
  remain_count  INT NOT NULL,
  per_limit     INT DEFAULT 1,                -- 每人限领
  start_time    DATETIME NOT NULL,
  end_time      DATETIME NOT NULL,
  status        TINYINT DEFAULT 1,
  created_at    DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 用户优惠券表 `user_coupons`
```sql
CREATE TABLE user_coupons (
  id          BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id     BIGINT NOT NULL,
  coupon_id   BIGINT NOT NULL,
  order_id    BIGINT,                         -- 关联订单（用掉时）
  status      TINYINT DEFAULT 0,              -- 0未使用 1已使用 2已过期
  received_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  used_at     DATETIME
);
```

#### 收藏表 `favorites`
```sql
CREATE TABLE favorites (
  id          BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id     BIGINT NOT NULL,
  product_id  BIGINT NOT NULL,
  created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY uk_user_product (user_id, product_id)
);
```

#### 商品评价表 `reviews`
```sql
CREATE TABLE reviews (
  id          BIGINT PRIMARY KEY AUTO_INCREMENT,
  product_id  BIGINT NOT NULL,
  user_id     BIGINT NOT NULL,
  order_id    BIGINT NOT NULL,
  sku_id      BIGINT,
  rating      TINYINT NOT NULL,              -- 1-5星
  content     TEXT,
  images      JSON,                          -- 评论图片
  is_anonymous TINYINT DEFAULT 0,
  reply        TEXT,                          -- 商家回复
  reply_at     DATETIME,
  created_at   DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 轮播图/广告表 `banners`
```sql
CREATE TABLE banners (
  id          BIGINT PRIMARY KEY AUTO_INCREMENT,
  title       VARCHAR(100),
  image       VARCHAR(500) NOT NULL,
  type        VARCHAR(20),                    -- product/category/url
  target_id   VARCHAR(100),                   -- 跳转目标
  sort_order  INT DEFAULT 0,
  is_show     TINYINT DEFAULT 1,
  start_time  DATETIME,
  end_time    DATETIME,
  created_at  DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

## 五、开发阶段规划

### 第一阶段：基础设施搭建 ⏱ 约 5 天

| 序号 | 任务 | 详情 |
|------|------|------|
| 1 | 项目初始化 | 创建 Vue 3 前端项目（Vite）、Node.js 后端项目（Express/Koa2）、MySQL 数据库 |
| 2 | 数据库设计 | 编写所有建表 SQL，建立索引，执行初始化脚本 |
| 3 | 后端架子 | 项目分层（Router/Controller/Service/Model）、统一响应封装、错误处理中间件 |

  "code": 200,
  "message": "success",
  "data": {
    "list": [...],

      "page": 1,

      "totalPages": 5
